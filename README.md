# 🏥 Hospital Management System

> A Python-based application to automate hospital operations like patient management, doctor records, and appointment scheduling.

---

## 🚀 Overview

The **Hospital Management System** is designed to replace manual hospital workflows with an efficient digital solution. It enables seamless management of patient data, doctor information, and appointment scheduling using Python and Excel-based storage.

This project demonstrates strong fundamentals in **Python, data handling, and real-world problem solving**.

---

## ✨ Key Features

### 🔐 Admin Module

* Secure login with password authentication
* Add, update, delete patient records
* Add, update, delete doctor records
* Manage appointments (book, edit, cancel)
* Automatic data saving to Excel

### 👤 User Module

* View hospital information
* View available doctors and patients
* Check appointment details

---

## 🛠️ Tech Stack

* **Language:** Python
* **Libraries:** Pandas, OpenPyXL
* **Storage:** Excel (.xlsx files)
* **Environment:** Python Console

## 🧠 System Workflow

1. Loads patient and doctor data from Excel
2. User selects mode: **Admin / User**
3. Admin manages records and appointments
4. User views hospital-related information
5. Data is updated and stored back in Excel
